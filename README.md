# LaundryKu

LaundryKu adalah website pengelolaan bisnis laundry berbasis HTML, CSS, JavaScript, dan Bootstrap 5.

## Fitur

- Dashboard
- Data Pelanggan
- Data Laundry
- Pembayaran
- Laporan
- Halaman Tentang
- Responsive Design
- Bootstrap 5
- Chart.js

## Teknologi

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Bootstrap Icons
- Chart.js

## Struktur Folder

```
website kuu/
│
├── assets/
│   ├── css/
│   ├── js/
│   ├── img/
│   └── icons/
│
├── pages/
│   ├── pelanggan.html
│   ├── laundry.html
│   ├── pembayaran.html
│   ├── laporan.html
│   └── tentang.html
│
├── index.html
├── style.css
├── main.js
└── README.md
```

## Cara Menjalankan

1. Buka folder proyek.
2. Jalankan file `index.html` menggunakan browser atau Spck Editor.
3. Gunakan menu navigasi untuk berpindah halaman.

## Tujuan

Website ini dibuat sebagai proyek mata kuliah dengan tema **Pengelolaan Bisnis Laundry Berbasis Web**.
